# 🎓 Student Management System

A full-stack web application to manage student records with CRUD operations and validation.

---

## 🚀 Tech Stack

* Angular (Frontend)
* Spring Boot (Backend)
* SQL Server (Database)

---

## ✨ Features

* Add, View, Update, Delete students
* Search students
* Form validation (email must contain `@`, city only letters)
* Error message: **"Please fix validation errors!"**
